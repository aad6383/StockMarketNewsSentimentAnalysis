# Stock Market News Sentiment Analysis

**Program:** UT Austin Post Graduate Program in Generative AI for Business Applications (AI Bootcamp)

## 📌 Project Overview
This project develops an AI-driven sentiment analysis system that automatically processes and analyzes financial news articles to determine market sentiment.  
The goal is to help financial analysts make more informed investment decisions by providing real-time insights derived from news data.

Using Natural Language Processing (NLP) and Generative AI techniques, the system classifies news articles as positive, negative, or neutral and summarizes overall market sentiment.

---

## 🎯 Objectives
- Build an automated pipeline for analyzing stock market news
- Apply NLP / LLM techniques for sentiment classification
- Demonstrate the use of Generative AI in financial analytics
- Generate insights that support investment decision-making
